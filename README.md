# RV32I Assembler in Common Lisp

[日本語](#日本語) | [English](#english)

---

## 日本語

### 概要

Common Lisp（SBCL）で実装したRV32Iアセンブラです。
テキスト形式のアセンブリファイル（`.s`）を読み込み、
RV32Iバイナリ（`.bin`）を生成します。

### 特徴

- RV32I基本命令セットを全てサポート
- ラベル対応（2パス方式）
- 即値・レジスタの範囲バリデーション
- SBCL上で動作

### 動作環境

- SBCL 2.6.4
- Arch Linux

### 使い方

```bash
# アセンブル
./asm.sh input.s
# → output.bin が生成される

# デフォルトはinput.s → output.bin
./asm.sh
```

### アセンブリ記法

```asm
; コメントはセミコロン
; レジスタはx0〜x31

    addi x1, x0, 0
    addi x2, x0, 10
loop:
    addi x1, x1, 1
    bne x1, x2, loop   ; ラベル使用可
    addi x17, x0, 10
    ecall
```

### 対応命令

| 型 | 命令 |
|----|------|
| R型 | ADD SUB AND OR XOR SLL SRL SRA SLT SLTU |
| I型 | ADDI ANDI ORI XORI SLTI SLTIU SLLI SRLI SRAI LW LH LB LHU LBU JALR |
| S型 | SW SH SB |
| B型 | BEQ BNE BLT BGE BLTU BGEU |
| U型 | LUI AUIPC |
| J型 | JAL |
| | ECALL |

### アルゴリズム

```
input.s
  ↓ parser.lisp（2パス）
    1パス目：ラベルアドレスを記録
    2パス目：ラベルを相対アドレスに変換
  ↓ encoder.lisp
    各命令をRV32Iバイナリにエンコード
  ↓ assembler.lisp
    リトルエンディアンでoutput.binに書き出し
```

## English

### Overview


A RV32I assembler implemented in Common Lisp (SBCL).
Reads text-format assembly files (`.s`) and generates RV32I binaries (`.bin`).

### Features

- Full RV32I base instruction set support
- Label support (2-pass assembler)
- Immediate value and register range validation
- Runs on SBCL

### Requirements

- SBCL 2.6.4
- Arch Linux (or any Linux with SBCL)

### Usage

```bash
# Assemble
./asm.sh input.s
# → output.bin is generated

# Default: input.s → output.bin
./asm.sh
```

### Assembly Syntax

```asm
; Comments start with semicolon
; Registers are x0-x31

    addi x1, x0, 0
    addi x2, x0, 10
loop:
    addi x1, x1, 1
    bne x1, x2, loop   ; Labels supported
    addi x17, x0, 10
    ecall
```

### Supported Instructions

| Type | Instructions |
|------|-------------|
| R | ADD SUB AND OR XOR SLL SRL SRA SLT SLTU |
| I | ADDI ANDI ORI XORI SLTI SLTIU SLLI SRLI SRAI LW LH LB LHU LBU JALR |
| S | SW SH SB |
| B | BEQ BNE BLT BGE BLTU BGEU |
| U | LUI AUIPC |
| J | JAL |
| | ECALL |

### Algorithm

```
input.s
  ↓ parser.lisp (2-pass)
    Pass 1: Record label addresses
    Pass 2: Resolve labels to relative addresses
  ↓ encoder.lisp
    Encode each instruction to RV32I binary
  ↓ assembler.lisp
    Write output.bin in little-endian format
```
